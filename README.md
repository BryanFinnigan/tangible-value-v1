# Tangible Value v1 Starter Kit

Clean Next.js App Router project with:
- Homepage with SEO/AEO-ready structure and SVG imagery
- Learn section
- First survey-course article
- Acquire section
- Amazon bullion sellers placeholder page
- Asset Dispatch newsletter UI

## Vercel settings

Framework Preset: Next.js  
Build Command: blank or `next build`  
Output Directory: blank  
Install Command: blank or `npm install`

Do not set Output Directory to `public` or `Next.js`.
